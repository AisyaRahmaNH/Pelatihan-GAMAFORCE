# Pelatihan GAMAFORCE

Repositori ini berisi materi dan kode latihan untuk pelatihan ESP32 dengan Python.

## Struktur Folder
- `src/`: Folder untuk kode Python yang akan diupload ke ESP32.
- `docs/`: Dokumentasi tambahan.
